## Технологии

- Next js 14 (app router)
- Tanstack Query
- Tailwind CSS
- axios
- Zustand

## Как запустить проект

- install Node js - https://nodejs.org/en
- npm install (alias -  npm i) - для загрузки всех пакетов (package.json)
- npm run dev (to launch a project in dev mode)
